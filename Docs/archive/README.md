# Archive

Status reports, fix logs, and superseded setup guides. Kept for history
only. Use the current docs in the parent directory instead.

## Layout

- `google-drive-setup.md` — Google Drive setup. The feature has been
  removed from the current build.
- `dropbox-setup.md`, `dropbox-quick-start.md`, `dropbox-ready.md` —
  earlier drafts of the Dropbox setup guide. Replaced by
  [`../dropbox-setup.md`](../dropbox-setup.md).
- `quick-start-gdrive.md` — Google Drive quick start. Feature removed.
- `quick-fix-*`, `fix-*`, `*-fix.md`, `error-409-fix.md` — bug-fix
  notes from the Google Drive integration work.
- `status-*`, `all-fixed.md`, `ready-to-test.md`, `setup-complete.md` —
  checkpoint reports from the same period.
- `deployment-checklist.md` — pre-Vercel deploy checklist. Replaced by
  [`../deploy.md`](../deploy.md).
- `persistent-login-features.md`, `production-login-fix.md`,
  `production-sync-fix.md` — feature notes for persistent login and
  sync behaviour. Now documented inline in `dropbox-setup.md`.
